Twitch Auto Claim

Описание (RU)

**Twitch Auto Claim** — это расширение для Firefox, которое автоматически забирает бонусные баллы канала (Channel Points) на Twitch во время просмотра стримов.

Расширение работает в фоновом режиме и автоматически нажимает кнопку «Получить бонус», когда она появляется.


 🚀 Возможности

Автоматический сбор Channel Points
Удобный GUI с включением / выключением
Кастомный интерфейс с фоном и логотипом Сохранение состояния (вкл/выкл)
Кнопка Donate
Поддержка русского и английского интерфейса Twitch


 ⚙ Как работает

Расширение:
Отслеживает появление кнопки бонуса
Автоматически нажимает её
Работает только на `https://www.twitch.tv/*`

 🔒 Безопасность

Расширение:
Не использует сторонние серверы
Не собирает данные
Не передаёт информацию
Работает только локально в браузере

Description (EN)

*Twitch Auto Claim* is a Firefox extension that automatically claims Twitch Channel Points bonuses while watching streams.

The extension runs in the background and clicks the "Claim Bonus" button whenever it appears.



🚀 Features

Automatic Channel Points claiming
Clean GUI with on/off toggle
Custom background and logo
Saved enabled/disabled state
Donate button
Supports Russian and English Twitch interface


⚙ How It Works

The extension:
Detects when the bonus button appears
Automatically clicks it
Works only on `https://www.twitch.tv/*`



🔒 Security

This extension:
Does not use external servers
Does not collect data
Does not transmit information
Runs fully locally in your browser
